Einrückung

Das einheitliche Einrücken von Zeilen ist für die Lesbarkeit des HTML-Codes sehr wichtig. Dadurch erkennen wir den Anfang und das Ende der einzelnen Blöcke. Gehen Sie zu den Einstellungen für den HTML-Editor

  1. File ⇒ Settings
  2. Editor ⇒ Code Style ⇒ HTML
  3. Tabs and Indents

HTML-Vorlage

Wenn Sie eine neue HTML-Datei erstellen, erzeugt Webstorm eine Grundstruktur. Diese Grundstruktur ist für unsere Zwecke noch nicht optimal: Die Sprache der Webseite auf englisch festgelegt: lang=“en”>
